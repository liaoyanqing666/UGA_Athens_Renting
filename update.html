Method 1

留下 Issue

  1. 适合希望公开记录问题、让后续修改过程可追踪的人。
  2. 如果你发现数据缺漏、价格变化、地图点位错误，最稳妥的方式是直接提 Issue。
  3. 在 Issue 里写清楚公寓名称、需要修改的字段，以及你看到的新信息来源。
前往提 Issue

Method 2

联系作者

  1. 适合不会用 GitHub、希望直接沟通的人。
  2. 建议把公寓名、要改的字段、修改后的值和信息来源一起发给作者，避免来回确认。
  3. QQ 邮箱：1793706453@qq.com；UGA 邮箱：sl64343@uga.edu。
访问作者主页

Method 3

自行更新表格后运行脚本

  1. 适合已经拿到新版 Excel、希望自己立即更新页面数据的人。
  2. 先 Fork 项目，在你自己的仓库里修改根目录的 26fall renting.xlsx。
  3. 运行 python .\scripts\build_data.py，重新生成 data/apartments.json。
  4. 确认页面数据正常后，提交 commit，再向主仓库发起 Pull Request。
