Update Data

更新 UGA Athens 租房数据

欢迎任何形式的更新建议！项目以静态网页的形式存在 Github 中，使用 excel 表格作为数据源。如果你想要指出错误，可以使用第一种和第二种方法留言；如果你希望自己更新并提交上来（这种情况下你会成为项目合作者），欢迎使用第三种方法，自己更新数据后提交 Pull Request。

返回主页 打开数据表格页

Method 1

留下 Issue

  1. 如果你发现数据缺漏、价格变化、信息错误等，可以直接在 Github 中提 Issue。
  2. 在 Issue 里写公寓名称、需要修改的字段等即可，或者留下任何你想说的话。
前往提 Issue

Method 2

联系作者

  1. 如果你完全不会使用 GitHub，或者希望与作者直接沟通或合作（甚至想跟作者加好友），可以直接联系作者。
  2. 如果想要更新数据，建议把公寓名、要改的字段、修改后的值等全部信息一起发给作者，避免来回确认。
  3. QQ 邮箱：1793706453@qq.com；UGA 邮箱：sl64343@uga.edu。
访问作者主页

Method 3

自行更新表格后运行脚本

  1. 这种方法会复杂一点，但可以让你成为项目合作者。
  2. 创建 Github 账号，点击项目右上角 Fork，将此项目 Fork 到你自己的账户中。
  3. 在你 Fork 的版本中，找到 data/26fall renting.xlsx 文件，clone 或者下载到本地编辑后再上传，更新租房数据。
  4. 注意编辑时请模仿其他行的编写逻辑，以及中英文符号的使用
  5. 使用 python 运行 build_data.py，重新生成数据库文件： data/apartments.json。
  6. 在本地打开 table.html和 index.html，确认数据正确显示。
  7. 将修改上传到你 Fork 的 Github 上（需要上传新的 26fall renting.xlsx 和 apartments.json）。
  8. 在 Github 上提交 Pull Request，等待项目维护者合并。
